# Suvidha — Universal Booking Portal (Demo V3)

This is an upgraded **Level 3** frontend demo: fully clickable pages + mock API (localStorage). No backend required — works entirely in browser.

## How to run
1. unzip the ZIP
2. cd universal-booking-frontend-v3
3. npm install
4. npm run dev
5. Open http://localhost:5173

## Features added in V3
- Full routing (Home, Categories, Category page, Provider profile)
- Booking flow using mock API and localStorage
- Mock login (store mobile in localStorage)
- My Bookings page with cancel (mock)
- Simple user dashboard
- Copy-ready code to wire up real API later
